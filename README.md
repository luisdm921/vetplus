# Veterinaria V2 — Firo Vet Premium Landing

Rediseño premium de la landing page para la clínica veterinaria Firo Vet.

## Stack

- **Next.js 15** + React 18
- **Tailwind CSS 3.4**
- **Framer Motion 11** (animaciones ligeras)
- **React Icons** (iconografía)
- **TypeScript**

## Ejecutar

```bash
npm install
npm run dev
```

Abre [http://localhost:3006](http://localhost:3006)

## Build

```bash
npm run build
npm start
```
